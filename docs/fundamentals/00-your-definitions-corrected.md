## 1) What is Angular?
Your version:
> Angular is a javascript framework used to build frontend web applications which are scalable and provides easy development experience to the community

Corrected:
Angular is a TypeScript-based frontend framework for building scalable web applications.

---

## 2) Components
Your version:
> components are the fundamental building block of any frontend framework. they help us seperate the ui and encapsulate the functionality into reusable blocks

Corrected:
Components are the fundamental building blocks of an Angular app. They separate UI and logic into reusable, independent parts.

---

## 3) Routing
Your version:
> Routing is a mechanism to navigate in the web application.

Corrected:
Routing is the mechanism used to navigate between views/components in a web application.

---

## 4) Forms
Your version:
> Form is another core part of a web application . It captures the user inputs provides it to the application

Corrected:
Forms are used to capture user input, validate it, and send it to the application.

---

## 5) Dependency Injection
Your version:
> dependcy injection , it is a concept where a component can receive the dependency (in another words , help from another service) using the class name as reference

Corrected:
Dependency Injection is a pattern where Angular provides required services to a class (like a component) instead of the class creating them itself.

---

## Memory shortcut
- Component = UI block
- Template = UI syntax
- Routing = page/view navigation
- Form = user input + validation
- DI = get service from Angular injector
